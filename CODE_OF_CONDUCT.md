# Code of Conduct

Riverbraid-Core is a reference implementation and governance artifact.
Participation in this repository is expected to remain respectful, technically grounded, and aligned with good-faith inquiry.

## Expected Behavior

- Engage with the code, documentation, and issues in a constructive and technically focused manner.
- Respect the stated architectural boundaries and governance constraints of the project.
- Disagree with ideas, not people.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks.
- Misrepresentation of the projectâ€™s intent, guarantees, or scope.
- Attempts to inject ideology, hype, or ego-driven narratives into technical discussions.
- Disruptive behavior that undermines the integrity of the repository.

## Enforcement

The maintainer reserves the right to moderate, remove, or restrict participation that violates this Code of Conduct.
This includes issues, pull requests, comments, and any other project-associated communication.

## Scope

This Code of Conduct applies within this repository and its official communication channels.

This project is not a product, a promise, or a movement.
It is a reference floor. This repository is maintained as a reference artifact. 

Maintainers may limit or decline engagement that does not align with its stated scope.
